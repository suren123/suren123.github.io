Hexo 常用命令

  • 奔跑的小鸭子

新建文章:

hexo n “文章名”
hexo new “文章名”

Subline Text 中快速预览

Alt + M

生成页面:

hexo g
hexo generate

生成并发布到Git:

hexo d -g

生成并发布到本地Server:

hexo s -g

清除:

hexo clean

运行本地服务器:

hexo server

主页:
http://localhost:4000

生成并发布到Git:

hexo d -g

发布到GitHub:

hexo deploy
